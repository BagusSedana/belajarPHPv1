Deskripsi Proyek
! Aplikasi To-Do List adalah alat manajemen tugas sederhana yang memungkinkan pengguna menambahkan, melihat, mengedit, dan menghapus tugas. Proyek ini mencakup fitur dasar CRUD (Create, Read, Update, Delete) menggunakan PHP dan MySQL !
==================================================================================================
#
#
==================================================================================================
Fitur Proyek
1.Tambah Tugas Baru: Formulir untuk menambahkan tugas baru.
2.Lihat Tugas: Daftar semua tugas yang ditambahkan.
3.Edit Tugas: Kemampuan untuk mengedit deskripsi tugas.
4.Hapus Tugas: Opsi untuk menghapus tugas dari daftar.
